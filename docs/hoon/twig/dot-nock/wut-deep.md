# `:deep  .?  "dotwut"`

`{$deep p/seed}`: test for cell or atom with Nock `3`.

## Produces

`&`, `%.y`, `0` if `p` is a cell; otherwise `|`, `%.n`, `1`.

## Syntax

Regular: *1-fixed*.

## Examples

```
~zod:dojo> :deep(~)
%.n
~zod:dojo> :deep([2 3])
%.y
```

```
~zod:dojo> .?(~)
%.n
~zod:dojo> .?([2 3])
%.y
```
