# `:fix  =:  "tiscol"` 

`{$fix p/(list (pair wing seed)) q/seed}`: change multiple legs in the subject.

## Expands to

```
:per(:keep(. p) q)
```

```
=>(%_(. p) q)
```

## Syntax

Regular: *jogging*, then *1-fixed*.

## Examples

```
~zod:dojo> :pin  a=[b=1 c=2]
           :fix  c.a  4
                 b.a  3
               ==
           a
[b=3 c=4]
```

```
~zod:dojo> =+  a=[b=1 c=2]
           =:  c.a  4
               b.a  3
             ==
           a
[b=3 c=4]
```
