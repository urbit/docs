# `!= "zaptis"`

`[%zpts p=hoon]`: make the Nock formula for a hoon.

## Produces

The Nock generated by `p`.

## Syntax

Regular: *1-fixed*.

## Examples

```
~zod:dojo> !=(20)
[1 20]
~zod:dojo> !=(~zod)
[1 0]
~zod:dojo> !=((add 2 2))
[8 [9 3.110.356 0 31] 9 2 [0 4] [7 [0 3] 1 2 2] 0 11]
```
