Parse space

Parses ASCII character 32, space.

Source
------

    ++  ace  (just ' ')

Examples
--------

    > (scan " " ace)
    ~~. 
    > `cord`(scan " " ace)
    ' '
    > (ace [[1 1] " "])

### `++bas`

Parse backslash

Parses ASCII character 92, the backslash. Note the extra `\` in the calling of
`bas` with [`++just`](/docs/hoon/library/2ec#++just) is to escape the escape
character, `\`.

Source
------

    ++  bas  (just '\\')

Examples
--------

    > (scan "\\" bas)
    ~~~5c.
    > `cord`(scan "\\" bas)
    '\'
    > (bas [[1 1] "\"])
    ~ <syntax error at [1 18]>
